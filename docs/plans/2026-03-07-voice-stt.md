# Voice-to-Text (STT) via Groq Whisper Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Automatically transcribe audio attachments (Discord voice messages, etc.) using Groq Whisper API before forwarding to Claude Code, replacing the local file path in the prompt with the transcribed text.

**Architecture:** Intercept in `src/llm-provider.ts` — before calling Claude SDK, scan the prompt for audio file paths matching `~/.claude-to-im/uploads/….(ogg|mp3|…)`, POST each file to Groq `/v1/audio/transcriptions`, and substitute the path with `[Voice message (transcribed): "…"]`. New logic lives in a dedicated `src/stt.ts` module. No upstream (`claude-to-im`) modification needed.

**Tech Stack:** Node.js built-in `fetch` (v18+), `FormData`/`Blob` (Node v18+), Groq REST API (`https://api.groq.com/openai/v1/audio/transcriptions`), `node:test` + `tsx` for tests.

**Step 3: Implement `src/stt.ts`**

Create `src/stt.ts`:

```typescript
/**
 * Speech-to-Text transcription via Groq Whisper API.
 *
 * Scans the prompt for audio file paths saved by the bridge adapters
 * (pattern: ~/.claude-to-im/uploads/<timestamp>_<name>.<audio_ext>)
 * and replaces each one with the Groq-transcribed text.
 */

import fs from 'node:fs';

const GROQ_TRANSCRIPTION_URL = 'https://api.groq.com/openai/v1/audio/transcriptions';
const GROQ_STT_MODEL = 'whisper-large-v3-turbo';

// Matches audio file paths anywhere in a string.
// The bridge saves files as: <uploads_dir>/<timestamp>_<safeName>.<ext>
// We match any absolute path ending with an audio extension.
export const AUDIO_PATH_REGEX =
  /\/[^\s"']+\/\d+_[^\s"']+\.(ogg|oga|mp3|mp4|wav|m4a|webm|flac)/g;

/**
 * Transcribe all audio file paths found in `prompt` using Groq Whisper.
 * Returns the prompt with paths replaced by transcription text.
 *
 * If `groqApiKey` is empty, or if an audio file is missing, or if the
 * Groq API returns an error, that path is left unchanged (graceful fallback).
 */
export async function transcribeAudioIfNeeded(
  prompt: string,
  groqApiKey: string,
): Promise<string> {
  if (!groqApiKey) return prompt;

  // Find all audio paths in prompt
  const matches = [...prompt.matchAll(AUDIO_PATH_REGEX)];
  if (matches.length === 0) return prompt;

  let result = prompt;

  for (const match of matches) {
    const filePath = match[0];

    // Check file exists
    try {
      fs.accessSync(filePath, fs.constants.R_OK);
    } catch {
      console.warn(`[stt] Audio file not found, skipping transcription: ${filePath}`);
      continue;
    }

    const transcription = await transcribeFile(filePath, groqApiKey);
    if (transcription === null) {
      // API error — leave path unchanged
      continue;
    }

    result = result.replace(filePath, `[Voice message (transcribed): "${transcription}"]`);
  }

  return result;
}

/**
 * POST a single audio file to Groq and return the transcribed text.
 * Returns null on any error (network, API, parse).
 */
async function transcribeFile(
  filePath: string,
  groqApiKey: string,
): Promise<string | null> {
  try {
    const fileBuffer = fs.readFileSync(filePath);
    const fileName = filePath.split('/').pop() ?? 'audio.ogg';
    const blob = new Blob([fileBuffer]);

    const form = new FormData();
    form.append('file', blob, fileName);
    form.append('model', GROQ_STT_MODEL);

    const response = await fetch(GROQ_TRANSCRIPTION_URL, {
      method: 'POST',
      headers: { Authorization: `Bearer ${groqApiKey}` },
      body: form,
    });

    if (!response.ok) {
      const body = await response.text();
      console.warn(`[stt] Groq API error ${response.status} for ${filePath}: ${body}`);
      return null;
    }

    const data = (await response.json()) as { text?: string };
    return data.text ?? null;
  } catch (err) {
    console.warn(`[stt] Transcription failed for ${filePath}:`, err);
    return null;
  }
}
```
